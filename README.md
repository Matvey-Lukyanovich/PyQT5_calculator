# PyQT5_calculator

### Operation:
##### "/" - Div (7/2=3.5)
##### "//" - Integer Div (7/2=3)
##### "%" - Mod (7/2=1)
##### "*" - Mul (7*2=14)
##### "+" - Sum (7+2=9)
##### "-" - Dec (7-2=5)
##### "√" - Sqrt (7√2 = 7^1/2))
##### "^" - Pow (4^3 = 64)
##### "^2" - x*x (7 = 49)
##### "=" - Get Result
##### "C" - Clear
